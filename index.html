Aarav M

A 15 year old independent developer

2014

My first experience with a language was at age 6-7 when I stumbled into a room with my father

I made a basic website with my father on an outdated version of Adobe Dreamweaver using HTML4 (both my father and I have evolved from that) and my passion for anything technological was fueled.

2015

For a birthday, I had received a coupon to a local bookstore with which I had bought a book on Scratch (I was 8) and Python.

It taught the basics of programming and kickstarted my passion in something other than black-on-aqua headers and Python is now my primary language.

2017/2019

I discovered the (at the time) very foreign language in 2017 known as Javascript and using online tutorials managed to memorize the document.getElementByIds and other DOM APIs required to make my code run.

In 2019, I made a couple of basic games in Q5.js, which I still regard as a good wrapper for WebGl, but have moved on from. Eventually, I moved onto Node.js and made a HTTP server. I learnt React (Native), in brief too, although function components were off my radar.

In 2020, I discovered Next.js and Typescript. I made a couple of unfinished Next.js projects before discovering the framework Remix, which combines the backend and frontend and implements features like nested routes.

2020

For my generation to learn C/C++ sounds like a step back, but I had interests beyond the high-level world. I was interested in how computers work on a fundamental level, and its nearly impossible to learn that from dynamically typed languages like JS and Python. With a handful of free time during the quarantine I sought to embark on the arduous journey of managing my own memory. I learnt the basic syntax and standard library using WSL as my compilation environment. Eventually I moved onto Assembly (look below) which opened my eye to abstracted away features like registers.

2021

After falling down the rabbit hole of low level programming, Assembly was the hard ground I landed on. Using nasm as my assembler, I learnt x86 Intel Assembly. It started with OS Development, and I learnt 16-bit Real Mode, then moved onto 32-bit Protected Mode, then I just moved to C at that point.

In late 2022, I finally plucked up the courage to write assembly for Linux. I made a few programs and plan to continue working on it.
